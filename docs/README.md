# Flex Documentation

Welcome to the official documentation for **Flex**.

This documentation site serves as the central knowledge base for understanding what Flex is, how it works, and how to contribute to the project.

Whether you're an executor developer, a curious tester, or a contributor helping document functions, you're in the right place!

!!! info "A standard, separate from sUNC"

    Flex is a distinct standard from sUNC, providing a space for Quality of Life (QoL) functions that don't fit sUNC's strict function inclusion criteria. This helps avoid bloating sUNC with convenience functions while offering proper testing and documentation for those that are still valuable.
